Supabase 데이터베이스 연결 미연결
학생 활동지와 세특 데이터를 Supabase에 저장합니다. 아래 설정을 한 번만 입력하면 이후 자동 연결됩니다.
Supabase 설정 방법 (최초 1회)
1. supabase.com → 무료 계정 가입 → New Project 생성
2. 좌측 메뉴 SQL Editor → 아래 SQL 붙여넣고 실행
-- 제출 데이터 테이블
create table submissions (
  id uuid primary key default gen_random_uuid(),
  name1 text, student_id1 text unique,
  class1 text, name2 text, student_id2 text,
  answers jsonb default '{}',
  reflection1 text, reflection2 text,
  seteuk1 text, seteuk2 text,
  submitted_at timestamptz default now()
);
-- 설정 테이블 (질문 저장용)
create table settings (
  key text primary key,
  value text
);
-- 익명 접근 허용
alter table submissions enable row level security;
alter table settings enable row level security;
create policy "allow all" on submissions for all using (true) with check (true);
create policy "allow all" on settings for all using (true) with check (true);
3. 좌측 메뉴 Project Settings → API
• Project URL 복사 → 위 URL 칸에 붙여넣기
• anon public 키 복사 → 위 Key 칸에 붙여넣기
4. 위의 "연결하기" 버튼 클릭 → 완료!

실행 방법 (단계별 안내)
1
위 파일 3개를 같은 폴더에 저장
바탕화면에 수업프로그램 폴더 만들고 세 파일을 그 안에 넣어두세요.
2
시작.bat 더블클릭
처음 실행 시 Node.js 포터블 버전을 자동으로 다운로드합니다 (약 30MB, 인터넷 필요).
두 번째부터는 인터넷 없이 바로 실행됩니다.
💡 Windows 보안 경고가 뜨면 "추가 정보" → "실행" 클릭
💡 방화벽 창이 뜨면 "허용" 클릭
3
검은 창에서 학생 주소 확인
서버가 켜지면 아래처럼 주소가 표시됩니다.
교사 주소 → http://localhost:3000
학생 주소 → http://192.168.1.5:3000
★ 위 학생 주소를 칠판에 적어주세요 ★
4
학생 주소를 칠판에 적기
학생들은 각자 컴퓨터 브라우저(크롬·엣지)에서 그 주소를 입력하면 접속됩니다.
설치 불필요, 주소 입력만으로 완료.
5
화면 방송 시작 (선택)
교사용 관리 → 화면 방송 탭 → 방송 시작 클릭
웹캠 또는 화면 공유를 선택하면 학생 화면에 실시간으로 전송됩니다.
주의사항
• 교사 컴퓨터와 학생 컴퓨터가 같은 와이파이(교내망)에 연결되어 있어야 합니다
• 수업 종료 후 검은 창에서 Ctrl+C를 눌러 서버를 종료하세요
• 학생들은 별도 설치 없이 브라우저만 있으면 접속 가능합니다
